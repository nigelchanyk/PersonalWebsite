AeroInvasion

Aero Invasion is a two-dimensional aircraft shooting game written in Java. It was originally a high school programming project coded by Vincent Tang and I. I updated a lot of features to the game for version 1.1 in order to enter the ShowCase Competition of University of Toronto.

Gameplay

The player is allowed to rotate freely, and is allowed choose from a list of weapons. Once the game starts, different enemies enter from all directions. The objective of this game is to defeat all hostile aircrafts, and to minimize damage received as much as possible.

Features

  • Can be played with a wiimote, a keyboard with a mouse, or a smartboard/touchscreen.
  • Self-made Star Wars opening crawl
  • LAN network versus mode

System Requirements

  • Minimum resolution of 1024 px x 768 px
  • Java Runtime Environment (JRE)
  • (Recommended) Rage Italic Font
  • (Optional) Wii remote controller and Bluetooth receiver

MIPS Debugger

A MIPS simulation tool written in Java for debugging purposes. The syntax of the MIPS assembly program is a simplified version of MIPS used in CS241 - Foundations of Sequential Programs.

Feature

  • Execute MIPS assembly program
  • Monitor changes in registers and memory
  • Detect syntax errors

System Requirements

  • Windows operating system
  • Java Runtime Environment (JRE)
